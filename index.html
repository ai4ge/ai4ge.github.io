ABOUT THE BOOK

教材简介

本书面向高校人工智能通识课程建设需求，系统介绍人工智能的基本概念、关键技术、典型应用及发展趋势，旨在帮助读者建立清晰的知识框架，培养智能时代所需的基本素养与跨学科视野。

全书按照“基础认知 → 技术领域 → 前沿发展”的逻辑组织。基础篇介绍人工智能的发展历程、计算思维与智能思维，以及机器学习基础；领域篇围绕计算机视觉、自然语言处理、大数据分析、具身智能和区块链展开；大模型前沿篇介绍大模型的基本原理、应用赋能、智能体、幻觉问题以及伦理与安全。

全书兼顾基础性、时代性与可读性，可作为高校人工智能通识课程教材，也可供对人工智能感兴趣的读者入门学习。

FEATURES

教材特色

从“讲什么、怎么讲、给谁讲”出发，兼顾知识体系、学生认知与课堂使用。

01

基础与前沿贯通

由基本概念与机器学习出发，延伸至大模型、智能体及伦理安全，回应人工智能发展的现实变化。

02

领域覆盖较为完整

涵盖视觉、语言、大数据、具身智能与区块链等重要方向，帮助学生形成整体认知。

03

面向公共课教学

注重案例引入与通俗讲解，各章相对独立，便于不同专业、不同学时灵活组合。

LEARNING PATH

三篇构成一条学习路径

第一篇

基础篇

认识人工智能从何而来，理解计算思维、智能思维与机器学习的基本逻辑。

第二篇

领域篇

走进视觉、语言、数据、具身智能与区块链，理解人工智能在不同场景中的展开方式。

第三篇

大模型前沿篇

理解大模型的生成原理、应用方式与现实边界，把握人工智能发展的新趋势。
